# Android Conversor de Base Numérica (CBN)

## en-us
Native app for Android. It was developed to help students who are starting to learn conversions from numerical bases.<br>
https://play.google.com/store/apps/details?id=ponciano.romulo.cbn<br>
**Language:** Java/XML<br>
**IDE:** Android Studio<br>
This project is discontinued because the idea is to refer you to multiplatform.
